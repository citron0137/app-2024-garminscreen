## 🛠️ 개발 환경
- Garmin Connect IQ SDK
- Visual Studio Code + Garmin 확장프로그램
- Monkey C 언어

## 🚀 빌드하기
1. VS Code에서 프로젝트 열기
2. `Ctrl+Shift+P` → "Monkey C: Build"
3. 시뮬레이터에서 확인!
